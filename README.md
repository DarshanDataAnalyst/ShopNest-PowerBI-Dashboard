# 📊 ShopNest Power BI Dashboard

An interactive data analytics project built using **Power BI** for the **ShopNest Store**.  
This project focuses on uncovering business insights from 9 datasets across customer behavior, sales, payments, delivery performance, and product feedback.

---

## ✅ Features

- 🔟 Top 10 Product Categories by Total Sales  
- 📦 Delayed Orders by Product Category  
- 📅 Monthly Comparison of On-Time vs Delayed Deliveries  
- 💳 Payment Method Analysis  
- ⭐ Product Rating Analysis (Top 10 & Bottom 10 Products)  
- 🗺️ State-wise Sales Performance  
- 📈 Seasonal (Quarterly) Sales Patterns  
- 💰 Yearly Revenue Trends & Growth

---

## 🛠 Tools & Skills Used

- **Power BI**: Data Modeling, DAX, Visualizations  
- **Excel/CSV**: Data preparation and structuring  
- **DAX Measures**: For calculated insights  
- **Data Visualization**: Bar, Column, Line, Area, Pie charts & Slicers

---

## 📁 Dataset Summary

This project uses 9 datasets:

- `customers_dataset`
- `orders_dataset`
- `order_items_dataset`
- `order_reviews_dataset`
- `order_payments_dataset`
- `products_dataset`
- `sellers_dataset`
- `geolocation_dataset`
- `product_categories_dataset`
